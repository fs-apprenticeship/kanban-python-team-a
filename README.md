# Kanban Python Project - Team A

Proof-of-concept Kanban board project for apprentices to manage and track tasks.

## Tech Stack
- **Backend:** Django 4.2
- **Frontend:** HTMX + Pico UI
- **Database:** SQLite (development)
- **Testing:** Pytest-Django

## Environmental set up 
1. Copy the example file: `cp .env.example .env`
2. Generate a secret key and add it to `.env`
